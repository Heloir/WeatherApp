<h4>Importe Localmente<h4>
  
Na sua IDE:

Digite:
```bash
git clone https://github.com/Heloir/WeatherApp
```
Depois iremos rodar o app, dessa forma:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Clique em http://localhost:3000, e você terá o projeto localmente ☕
